# 🌤️ Weather App Pro

A modern weather application built with HTML, CSS, and JavaScript.

## 🚀 Live Demo

https://mohsenmoghadam1990.github.io/weather-app-pro/

---

## ✨ Features

- 🔍 Search weather by city
- 📍 Current location weather
- 🌙 Dark / Light mode
- ⭐ Favorite cities
- 📅 5-day weather forecast
- 💨 Wind speed
- 💧 Humidity
- 🌡️ Temperature
- ⏰ Live clock
- 📱 Responsive design
- ⚡ Progressive Web App (PWA)

---

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript
- OpenWeather API
